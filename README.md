# 人材系LP - バイト求人サイト

シンプルで使いやすいバイト求人管理システムです。求人の検索・閲覧、応募フォーム、管理者による応募管理機能を提供します。

## 📋 機能一覧

### ユーザー向け機能

- **求人一覧表示** (`index.html`)
  - 40件以上のバイト求人を表示
  - キーワード検索（バイトの種類で検索）
  - 地域フィルタ（東京都、神奈川県、埼玉県、千葉県、大阪府）
  - ページネーション（20件/ページ）
  - 求人カードに写真、アピールポイント、コメントを表示

- **求人詳細ページ** (`detail.html`)
  - 求人の詳細情報を表示
  - 応募ボタンから応募フォームへ遷移

- **応募フォーム** (`apply.html`)
  - 応募情報の入力・送信
  - デモデータの自動入力機能
  - フォームバリデーション

- **プライバシーポリシー・利用規約**
  - `privacy.html`: 個人情報の取り扱いについて
  - `terms.html`: 利用規約

### 管理者向け機能 (`admin.html`)

- **応募管理タブ**
  - 未連絡の応募一覧を表示
  - 連絡済みマーク機能
  - 連絡メモの保存
  - 応募の削除
  - ページネーション（20件/ページ）

- **応募一覧タブ**
  - 連絡済みの応募を店舗別にグループ化して表示

- **バイト求人管理タブ**
  - 求人の一覧表示
  - キーワード検索機能
  - 求人の新規追加・編集・削除
  - ページネーション（20件/ページ）

## 🚀 使い方

### セットアップ

1. すべてのファイルを同じディレクトリに配置します
2. `index.html` をブラウザで開きます
3. 初回起動時、デフォルトの求人データ（40件）が自動的に `localStorage` に保存されます

### データの保存

- すべてのデータ（求人情報、応募情報）はブラウザの `localStorage` に保存されます
- ブラウザのキャッシュをクリアすると、データがリセットされます

### 管理者ページへのアクセス

- `index.html` のヘッダーにある「管理者」リンクからアクセスできます
- または直接 `admin.html` を開くこともできます

## 🔧 主な機能の実装詳細

### 検索・フィルタ機能

### カーソル表示の最適化

- 入力フィールドのカーソル色を確実に表示
- 動的に追加される要素にも自動適用

## 📱 レスポンシブブレークポイント

- **デスクトップ**: 769px以上
- **タブレット**: 481px - 768px
- **モバイル**: 480px以下

## 🐛 既知の制限事項

- データは `localStorage` に保存されるため、ブラウザを変更するとデータが引き継がれません
- 外部画像URL（Unsplash等）を使用しているため、インターネット接続が必要です
- 配送ドライバーの画像はSVGデータURIを使用しています
